知识点

display 属性

  1. block：块级元素，一个块级元素会新开始一行并尽可能撑满容器。常见块级元素：div、p、form、header、footer、section等。
  2. inline：行内元素，一个行内元素可以在段落中包裹一些文字并不会打乱段落的布局。常见行内元素：span、a等。
  3. none：把 display 设置为 none 之后，不会保留元素本该显示的空间。visibility: 4. hidden 还会保留。
  4. inline-block：将对象呈递为内联对象，但是对象的内容作为块对象呈递。

https://developer.mozilla.org/en-US/docs/Web/CSS/display http://www.cnblogs.com/KeithWang/p/3139517.html

水平居中（margin: 0 auto）

示例代码：

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>水平居中</title>
    <style>
        .center {
            max-width: 1000px;
            margin: 0 auto;
            background: green;
        }
    </style>
</head>
<body>
    <div class="center">hello world</div>
</body>
</html>

position

static：默认。一个 static 元素表示它不会被“positioned”，一个 position 属性被设置为其他值的元素表示它会被“positioned”。 relative：相对于自身定位。使用 top、right、bottom 和 left 属性可以让元素偏离其正常位置 absolute：相对于最近的“positioned”祖先元素。如果绝对定位（position属性的值为absolute）的元素没有“positioned”祖先元素，那么它是相对于文档的 body 元素，并且它会随着页面滚动而移动。记住一个“positioned”元素是指p osition 值不是 static 的元素。 fixed：相对于视窗来定位，这意味着即便页面滚动，它还是会停留在相同的位置。和 relative 一样， top 、 right 、 bottom 和 left 属性都可用。

示例代码：导航栏

<!doctype html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>nav</title>
    <style>
        header {
            max-width: 1000px;
            margin: 0 auto;
            background: green;
        }

        .man-nav {
            position: relative;
        }

        .ul-links {
            position: absolute;
            right: 0;
            top: 50%;
            margin-top: -10px;
        }

        .ul-links li {
            list-style-type: none;
            float: left;
            margin-left: 8px;
        }

        .ul-links:after {
            clear: both;
        }
    </style>
</head>
<body>
<header>
    <nav class="man-nav">
        <div class="logo">MyLogo</div>
        <ul class="ul-links">
            <li><a href="">nav1</a></li>
            <li><a href="">nav1</a></li>
            <li><a href="">nav1</a></li>
        </ul>
    </nav>
</header>
</body>
</html>
